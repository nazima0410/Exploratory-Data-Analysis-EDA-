# Exploratory-Data-Analysis-EDA-
I performed Exploratory Data Analysis (EDA) using Python. I used Pandas, Matplotlib, and Seaborn to visualize the data and identify patterns, trends, and relationships between variables. This helped in understanding the dataset better and drawing meaningful insights.
I performed Exploratory Data Analysis (EDA) on a marketing campaign dataset to understand customer behavior and identify where marketing budget should be focused.
Instead of building a machine learning model, this analysis focused on the “why” behind customer responses.

🔍 Key Insights:
 • High-income customers showed higher campaign response rates
 • Customers aged 18–30 were more engaged
 • Frequent web purchasers were more likely to convert
 • High-spending customers responded better to campaigns

📊 Based on the analysis, marketing budget should prioritize:
 ✔ High-income segments
 ✔ Digitally active customers
 ✔ High-spending customer groupS
